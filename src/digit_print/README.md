
1. You've to display the digits of a number.
2. Take as input "n", the number for which digits have to be displayed.
3. Print the digits of the number line-wise.

## Constraints

```
1 <= n < 10^9
```

## Format

**Input**

```
"n" where n is any integer.
```

**Output**
```
d1
d2
d3
... digits of the number
```

## Example

**Sample Input**
```  
65784383
```  

**Sample Output**
```  
6
5
7
8
4
3
8
3
```
